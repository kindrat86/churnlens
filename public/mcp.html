MCP Server

ChurnLens — MCP Server

SaaS revenue quality scoring and due diligence tool.

One endpoint. Stateless HTTP transport. No auth required. Paste it into any MCP client:

https://churnlens.site/api/mcp

Install in Claude Desktop

Add to claude_desktop_config.json:

{
  "mcpServers": {
    "churnlens": {
      "command": "npx",
      "args": ["mcp-remote", "https://churnlens.site/api/mcp"]
    }
  }
}

Install in Cursor

Add to .cursor/mcp.json:

{
  "mcpServers": {
    "churnlens": {
      "url": "https://churnlens.site/api/mcp"
    }
  }
}

Inspect

npx @modelcontextprotocol/inspector https://churnlens.site/api/mcp

Available Tools (3)

  • calculate_churn_rate — Calculate churn rate, GRR, and NRR from MRR data.
  • get_saas_benchmark — Get SaaS benchmark data for churn rates and NRR.
  • analyze_revenue_concentration — Assess revenue concentration risk using HHI.
